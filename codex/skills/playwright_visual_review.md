# Playwright Visual Review

목표: 빌드 통과가 아니라 실제 화면 품질을 확인한다.

## 필수 시나리오

- 데스크톱: 1440px 이상
- 태블릿: 1024px 전후
- 모바일: 430px 전후

## 화면별 점검

- 사이드바와 본문 위계가 유지되는가
- 제목, 설명, 배지, 버튼이 너무 빽빽하거나 너무 멀지 않은가
- 텍스트 잘림, 오버플로, 이중 스크롤이 없는가
- 빈 상태가 허전하지 않고 다음 행동을 알려주는가
- 포커스 링이 배경에서 보이는가

## 결과물

- 핵심 화면 스크린샷
- 발견한 문제와 수정 여부
- 남은 모바일/태블릿 타협점
